Graph Terminology and Notation

An undirected graph is a pair $G=(V,E)$ where $V$ is a finite non-empty set of vertices or nodes and $E\subseteq 2^V$ is a finite (possibly empty) set of edges consisting only of subset of cardinality 2 (two nodes form an edge).

We use the notation $u\mathrm{-}v$ (equivalently, $v\mathrm{-}u$ in undirected graph) for both edges {$u,v$} and to say $u$ and $v$ are linked by an edge.
So that $G$ can be represented in two ways:

$G = (\{1,2,3,4\}, \{\{1,2\}, \{1,3\}, \{2,3\}, \{2,4\}\})$


or

$$G = (\{1,2,3,4\}, \{1\mathrm{-}2, 1\mathrm{-}3, 2\mathrm{-}3, 2\mathrm{-}4\})$$


We will say that the edge $u\mathrm{-}v$ is incident to either of its endpoints $u$ and $v$.
Two vertices such that $u\mathrm{-}v$ are called adjacent(or neighbors).
Note that we now only deal with simple graphs where there are no “loops” or “parallel edges”.

The degree of a vertex, $deg(u)$, is the number of neighbors of $u$ (the number of vertices adjacent to $u$).
A vertex of degree 0 is called isolated.

The Handshaking Lemma

The sum of the degrees of all nodes in a graph is twice the number of edges, that is

$\sum_{v\in V} deg(v) = 2 |E|$

Corollary: In any graph there are an even number of vertices of odd degree.
The proof of this corollary can be found at The Proof of Handshake Lemma’s Corollary by Induction.

Special Graphs

Edgeless graphs

An edgeless graph $G = (V, E)$ has some vertices but no edges. Therefore, $E = \emptyset$ hence $|E| = 0$.

Complete graphs

A complete graph on $n\geq 1$ vertices, notation $K_n$, has edges between any two vertices.
Therefore, $K_n = (V,E)$ where $|V| = n$ and $|E| = \{\{u,v\}|u,v\in V\}.$

The number of edges in $k_n$ is the number of unordered pairs $\binom{n}{2} = \frac{n(n-1)}{2}$.

Path graphs

A path graph on $n\geq 1$ vertices, notation $P_n$, has $n$ vertices and $n-1$ edges arranged “in a row”. A path graph on $n$ vertices is a graph isomorphic to $P_n$. Hence, a path graph of length $l$ is a graph isomorphic to $P_{l+1}.$

Note that for $n\geq3$ we have two vertices of degree 1 in $P_n$ and the rest have degree 2.

Cycle graphs

A cycle graph on $n\geq 3$ vertices, notation $C_n$, has $n$ vertices and $n$ edges arranged “in a circle”. Note that in cycle graphs all vertices have degree 2.

Grid graphs

Let $m,n\geq 1$. An $m\times n$ grid graph has $m$ rows of $n$ vertices. Each vertex is linked by an edge to the vertices “closest” to it.

Walks and Paths

Walks

A walk is a non-empty sequence of vertices consecutively linked by edges: $u_0, u_1, \ldots, u_k$ such that $u_0\mathrm{-} u_1\mathrm{-} \ldots\mathrm{-} u_k$. This walk is form $u_0$ to $u_k$ (the endpoints of the walk) and $u_0$ and $u_k$ are connected by this walk. The length of this walk is the number $k$ of edges.

In this graph, $1\mathrm{-}2\mathrm{-}4$ is a walk of length 2.

Paths

A path is a walk in which all the vertices are distinct.
Walks of length 0 are paths.
Walks of length 1 are already paths because we cannot have loops in simple graph.
$u\mathrm{-}v\mathrm{-}u$ is a walk of length 2 but not a path. For example, $3\mathrm{-}2\mathrm{-}4$ in the graph above is a path of length 2.

Proposition: If $u_0\mathrm{-} u_1\mathrm{-} \ldots\mathrm{-} u_{n-1}\mathrm{-} u_{n}$ is a walk of length $n\geq3$ such that $u_0\neq u_n$, then there exist vertices $v_1, \ldots,v_m$ such that $u_0\mathrm{-} v_1\mathrm{-} \ldots\mathrm{-} v_m\mathrm{-} u_{n}$ is a path of length at most $n$. (Where there is a walk, there is a path that is not longer.)

Stronger Proposition: If $u_0\mathrm{-} u_1\mathrm{-} \ldots\mathrm{-} u_{n-1}\mathrm{-} u_{n}$ is a walk of length $n\geq3$ such that $u_0\neq u_n$, then there exist vertices $v_1, \ldots,v_m$ such that $u_0\mathrm{-} v_1\mathrm{-} \ldots\mathrm{-} v_m\mathrm{-} u_{n}$ is a path whose sequence of nodes and edges is a subsequence of the sequence of nodes and edges of $u_0\mathrm{-} u_1\mathrm{-} \ldots\mathrm{-} u_{n-1}\mathrm{-} u_{n}$.
Here, “subsequence” preserves order, but it does not necessarily consist of consecutive elements.

Connected Components

Connectivity

Two vertices $u$ and $v$ of a graph $G=(V,E)$ are said to be connected, notation $u\mathrm{-}\ldots\mathrm{-}v$, if there exists some walk with endpoints $u$ and $v$. We call $\mathrm{-}\ldots\mathrm{-}$ the connectivity relation.
Connectivity is an equivalence relation, which means that connectivity is reflexive, symmetric and transitive.

Connected Components

A connected component of a graph $G = (V,E)$ is a set of vertices $C\subseteq V$ such that:

  • any two vertices in $C$ are connected, and
  • there is no strictly bigger set of vertices $C \subsetneq C’ \subseteq V$ such that any two vertices in $C’$ are connected.
    We say that $C$ is a maximally connected set of vertices of $G$.

In this graph we have two connected components:$\{1,2,3,4\}$ and $\{5,6,7\}$.

Partition

Clearly, every vertex is in some connected components, at worst by itself.
Proposition: Any two distinct connected components are disjoint.
The connected components form a partition of the vertices.

Connected graphs

A graph in which any two vertices are connected is called connected, other wise it is called disconnected.
A connected graph has exactly one connected component. A disconnected graph has tow or more connected components.
Complete graphs, $K_n$, path graphs, $P_n$, and cycle graphs, $C_n$, are all connected.
In edgeless graphs, each vertex forms a separate connected component. Thus, edgeless graphs with two or more vertices are disconnected, while 1-vertex graphs are connected.
The $m\times n$ grid graphs are connected.

Counting Connected Components

Consider a graph $G = (V,E)$ and denote by $CC$ the set of connected components of $G$. Shorthand: $cc$ for “connected component.”
Proposition: max($|V| - |E|$, 1) $\leq|C|\leq|V|$.
Proposition: In any graph $G = (V,E)$ we have $|E|\geq|V|-|CC|$.
Proposition: $|E| \leq \binom{|V|}{2}$.
Corollary: Every connected graph with $n$ vertices has $n-1$ or more edges.

Graph isomorphism

Two graphs $G_1 = (V_1,E_1)$ and $G_2 = (V_2,E_2)$ are isomorphic, notation $G_1\simeq G_2$, when there is a bijection $\beta : V_1 \rightarrow V_2$ such that for any $u_1,v_1 \in V_1$ we have $u_1 \mathrm{-}v_1 \in E_1$ iff $\beta(u_1)\mathrm{-}\beta(v_1)\in E_2$.

$G_1\simeq G_2$ by the bijection $4\mapsto 5, 2\mapsto 6, 1\mapsto 7, 3\mapsto 8$
Proposition: Any two complete graphs are isomorphic iff they have the same number of vertices. The same holds for path, cycle, and edgeless graphs.
Any two $m\times n$ grids are isomorphic, as well as isomorphic to any $n\times m$ grids.

Subgraph

A graph $G_1 = (V_1,E_1)$ is a subgraph of the graph $G_2 = (V_2,E_2)$ when $V_1 \subseteq V_2$ and $E_1 \subseteq E_2$.

Here, $G_1$ is a subgraph of $G_2$.

Induced subgraphs

If $G = (V,E)$ is a graph and $V’ \subseteq V$ is a set consisting of some of $G$’s nodes, the subgraph of $G$ induced by $V’$ is the graph $G’=(V’,E’)$ where $E’$ consists of all the edges of $G$ whose endpoints are both in $V’$.

$G’$ is the subgraph of $G$ induced by the subset of vertices $\{1,2,3\}$.
The subgraph of $G$ induced by $\{2, 3, 4\}$ is path graph of length 2.
The subgraph of $G$ induced by $\{1, 3, 4\}$ is disconnected.

Cycles

Closed Walks and Cycles

A closed walk is a walk in which the first and the last vertex are the same.
A cycle is a closed walk of length at least 3 in which all nodes are pairwise distinct, except for the last and the first.
The length of the cycle is the length of the closed walk.
When we count the cycles of length $n$ in a graph $G$, we count in facet the subgraphs of $G$ that are cycle graphs on $n$ vertices.

In graph $G$, $1\mathrm{-}2\mathrm{-}3\mathrm{-}4\mathrm{-}2\mathrm{-}1$ is a closed walk but not a cycle. $2\mathrm{-}3\mathrm{-}4\mathrm{-}2$ is a cycle of length 3.
